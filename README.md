# 📖 [Blog App] <a name="about-project"></a>


**[Blog App]**  will be a classic example of a blog website. This is a fully functional website that will show the list of posts and empower readers to interact with them by adding comments and liking posts.

### Setup

Clone this repository to your desired folder:


```sh
  cd my-folder
  git clone git@github.com:mperezbazan/blog-app.git
```

### Install

Install this project with:

```sh
  cd blog-app
  bundle install
  rails db:create
  rails db:migrate
```


### Usage

To run the project, execute the following command:


```sh
  rails server
```


### Run tests

To run tests, run the following command:


```sh
  rspec
```
